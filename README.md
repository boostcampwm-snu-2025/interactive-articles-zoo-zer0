## 
만약 새로운 기사를 작성하고 싶으면:
1) create a new folder <br>
2) add article.md and meta.js<br>
3) write the article<br>
4) register it in articles.js<br>
(간단하게 올릴 수 있음)




## 서비스 목적
재미 없을 수도 있는 데이터 관련 기사를 인터랙티브하고 미적으로 아름답게 전달하기. 새로운 기사 작성하기 편리하게 만들기.
